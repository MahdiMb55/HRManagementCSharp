# راهنمای UI/UX

رابط کاملاً فارسی، راست‌به‌چپ، روشن، کم‌حرکت و مبتنی بر کنترل‌های استاندارد WinForms است. `MainForm` منوی کناری ثابت و محتوای cacheشده دارد. جزئیات کارمند در یک پنجره non-modal قابل‌بازاستفاده باز می‌شود و تب‌ها در اولین انتخاب load می‌شوند.

فهرست کارکنان `DataGridView` فقط‌خواندنی با اندازه صفحه 25/50/100، انتخاب چندردیفی، مرتب‌سازی سمت SQLite، debounce سیصد میلی‌ثانیه و پنل خلاصه بدون تصویر داخل grid است. `Ctrl+N` افزودن، `Ctrl+F` تمرکز جست‌وجو و `Enter` بازکردن رکورد را انجام می‌دهند.

فرم افزودن/ویرایش ساختار مشترک، Save صریح، `ErrorProvider`، ترتیب Tab، حالت مشغول، جلوگیری از اجرای دوباره و هشدار تغییرات ذخیره‌نشده دارد. تاریخ با adapter جلالی نمایش داده و به `DateOnly` میلادی تبدیل می‌شود؛ culture سراسری تغییر نمی‌کند.

Vazirmatn از resource خصوصی یک‌بار load می‌شود. layout با `TableLayoutPanel`، `FlowLayoutPanel`, `SplitContainer`, Dock و Anchor و high-DPI ساخته می‌شود. افکت‌های blur، acrylic، Mica، shadow پرهزینه، animation پیوسته و ساخت دوباره صفحه در ناوبری ممنوع‌اند.
